# 📊 E-commerce Sales Dashboard (Power BI)

## 📌 Project Overview
This project is an interactive Power BI dashboard built to analyze and track e-commerce sales performance.

## 🚀 Key Features
- Monthly profit analysis
- Category-wise sales insights
- State-wise revenue distribution
- Customer-level analysis
- Payment mode breakdown

## 🛠 Tools & Technologies
- Power BI
- Data Modeling
- DAX (Basic)
- Data Visualization

## 📚 Learnings
- Created interactive dashboards using slicers and filters
- Built relationships and data models
- Used different visualization techniques
- Improved analytical thinking
